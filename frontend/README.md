# Frontend – Game Swap

Aplicación frontend de la plataforma web para la compra, venta e intercambio de videojuegos físicos.

Desarrollada con Angular y conectada al backend mediante una API REST.

## Tecnologías
- Angular CLI: 19.2.19
- Node 18.20.8
- TypeScript
- Angular Material
- HTML / CSS

## Instalación
Desde la carpeta `frontend` ejecutar:

```bash
npm install
